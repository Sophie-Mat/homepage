# Homepage
My first personal website. After all, it is just a résumé.

This project was developed as part of the **Binary Studio Academy** program.

## Overview
This repository contains a personal homepage constructed using standard web technologies (HTML, CSS, and JavaScript). It is designed to showcase my skills and act as a digital resume for the Academy.

## Project Structure
- `index.html`: The main structural document.
- `styles.css`: Custom styling and layout.
- `script.js`: Interactive functionality.
- `assets/`: Images and other static assets.
